# Gramkafe Coffee Website

This project is a Next.js + React + Tailwind website for Gramkafe.

## Local development

Install dependencies and run the dev server:

```bash
npm install
npm run dev
```

Open http://localhost:3000 in your browser for live preview.

## Production build

```bash
npm run build
```

## Deployment

This repository deploys automatically to GitHub Pages using the workflow in `.github/workflows/deploy.yml` whenever code is pushed to `master`.

Live site: https://denysovski.github.io/Gramkafe/
